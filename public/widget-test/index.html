How to generate authHash for testing

Run this in Node.js (or any server environment) using your Unchurnly app secret:

const crypto = require('crypto') const customerId = 'cus_xxxxxxxxxxxxxxxx' const appSecret = 'YOUR_APP_SECRET' // from Dashboard → Installation const hash = crypto .createHmac('sha256', appSecret) .update(customerId) .digest('hex') console.log(hash)

Paste the output into the Auth Hash field above, then click Cancel Subscription.
